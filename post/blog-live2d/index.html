    参考地址：https://docs.paul.ren/pio/

    模型资源：https://mx.paul.ren/page/3/

    由于是独立使用

    1. 引用 CSS 资源

      在你网站的 head 标签内引用本项目的 CSS 样式表文件：

      <!-- 引用看板娘交互所需的样式表 -->
<link href='你的站点路径/static/pio.css' rel='stylesheet' type='text/css'/>

    2. 引用 JS 资源

      <!-- 引用 Live2D 核心组件 -->
<script src='你的站点路径/static/l2d.js'></script>
<!-- 引用看板娘交互组件 -->
<script src='你的站点路径/static/pio.js'></script>

    3. 插入 DOM 内容

      <div class="pio-container {自定位置}">
    <div class="pio-action"></div>
    <canvas id="pio" width="{看板娘宽度}" height="{看板娘高度}"></canvas>
</div>
      参数名 允许值
      自定位置 left right
      看板娘宽度 整数，不带 px，即 HTML 写法
      看板娘高度 整数，不带 px，即 HTML 写法

    4. 设置启动器

      var pio = new Paul_Pio({
    "mode": "fixed",
    "hidden": true,
    "content": {
        "welcome": ["欢迎来到保罗的小宇宙！", "今天天气不错，一起来玩吧！", "博主每天都有些折腾记录，欢迎前往他的小窝阅读~"],
        "custom": [
            {"selector": ".comment-form", "text": "欢迎参与本文评论，别发小广告噢~"},
            {"selector": ".home-social a:last-child", "text": "在这里可以了解博主的日常噢~"},
            {"selector": ".post-item a", "type": "read"},
            {"selector": ".post-content a, .page-content a", "type": "link"}
        ]
    },
    "night": "single.night()",
    "model": ["https://paugram.com/usr/plugins/Pio/models/yuan/model.json"]
});

      注意把外链的model替换成下载到本地的文件

      开始修改Gride主题Song的模板文件

      因为时不使用插件，因此只是采用静态资源文件部署。

      ​ 没什么好说的，下载老婆（大雾）和泉纱雾，把解压缩的文件丢到song主题的song/assets/media/model下，model文件夹是我自己建的。其他不用管，可以把model的外联改为"model": ["/media/model/model.json"]。
      将demo中的static资源文件l2d.js、pio.css、pio.js放到可以访问的文件夹中，我放到了song/assets/media/scripts目录下，因为song主题编译的时候这些静态文件都会被复制到相应的media文件夹中，因此引入时写为

      1. 引用 CSS 资源

        修改templates\includes\head.ejs中

           <!-- 引用看板娘交互所需的样式表 -->
   <link href='/media/scripts/pio.css' rel='stylesheet' type='text/css'/>

      2. 设置启动器 footer.ejs中添加引用的 JS 资源和插入的DOM内容，在</footer>标签前添加即可

           <!-- 看板娘-->
    <div class="pio-container right">
        <div class="pio-action"></div>
        <canvas id="pio" width="250" height="450"></canvas>
    </div>
    <!-- 引用 Live2D 核心组件 -->
    <script src='/media/scripts/l2d.js'></script>
    <!-- 引用看板娘交互组件 -->
    <script src='/media/scripts/pio.js'></script>
    <script>
        var pio = new Paul_Pio({
            "hidden": true,
            "content": {
                "welcome": ["欢迎来到布夕的小宇宙！", "今天天气不错，一起来玩吧！", "博主每天都有些折腾记录，欢迎前往他的小窝阅读~"],
                "custom": [
                    {"selector": ".comment-form", "text": "欢迎参与本文评论，别发小广告噢~"},
                    {"selector": ".home-social a:last-child", "text": "在这里可以了解博主的日常噢~"},
              {"selector": ".post-item a", "type": "read"},
                    {"selector": ".post-content a, .page-content a", "type": "link"}
                ]
            },
            "night": "single.night()",
            "model": ["/media/model/model.json"]
        })
    </script>

    如此便完毕了。
