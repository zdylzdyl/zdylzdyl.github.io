    • Linux系统的一些图标记录
    2020-04-27 / 2 min read

    ubuntu16.04系统默认很多的图标快捷方式都是在目录/usr/share/applications
    对于一些自带的软件就不需要自己编辑了。
    备份一些软件的快捷方式：

    
[Desktop Entry]
Encoding=UTF-8
Name=LXTerminal
Name[zh_CN]=LX 终端
GenericName=Terminal
GenericName[zh_CN]=终端
Comment=Use the command line
Comment[zh_CN]=使用命令行
Comment[zh_TW]=使用命令列
TryExec=lxterminal
Exec=lxterminal
Icon=lxterminal
Type=Application
Categories=GTK;System;TerminalEmulator;
Keywords=console;command line;execute;

[Desktop Entry]
Keywords=editor
Name=Leafpad
Comment=Simple text editor
Comment[zh_CN]=简易文字编辑器
Exec=leafpad %f
Icon=leafpad
Terminal=false
Type=Application
MimeType=text/plain
Categories=GTK;Utility;TextEditor;

[Desktop Entry]
Name=gridea
GenericName=gridea
Exec=gridea

Type=Application
