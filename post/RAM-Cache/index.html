    由于笔记本内存升级到了16GB，而日常使用的量并不多，不像8GB的时候经常跑满，于是折腾了内存盘Ramdisk。
    顾名思义，内存盘，用内存当作硬盘使用，搜索了一下，很多软件都不错，随手选了一个虚拟光驱内存盘UltraRAMDisk1.65做内存盘使用
    一开始机械还在，使用机械和内存盘混合做虚拟盘，但是机械接口不太好，只好卸下机械了。考虑到唯一的硬盘是固态，本来读写就是避免固态的频繁读写，于是干脆整了一个纯粹的内存盘，目前先放了4GB，还算可以。

    1. 动态分配内存，这样不容易炸内存
    2. 驱动器号R，看起来也直观，卷标就是RAM
    3. 更改IE缓存
    4. 变更环境变量TEMP到内存盘
    5. 修改chrome的缓存到内存盘
    6. 由于我没有交换文件，有交换文件建议优先放在机械给固态续命

    但是只有这些怎么够？那么多临时文件，都放在内存盘该多好？
    于是 win+R 输入 regedit 开启注册表
    计算机\HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Explorer\Shell Folders
