    • LG V30 US998电信联通网络破解配置
    2020-06-06 / 4 min read

    被种草久了，于是入手了LG V30 US998美版吧，咸鱼入手的，价格420单机无配件

    入手配置是已经Root和BL解锁的，面具模块也是都好了，可惜太费电了，而且我对安全要求比较高，ROOT是不可能ROOT的，ROOT了我又没有什么好处，相反，ROOT获取的功能中我需要的，我基本都能靠ADB获取到。嗯，前情就酱紫。

    到手重置系统，发现已有TRWP，重置后，可以插卡电信卡，可以短信，我就大意了，没仔细检查就收货了。上一个洋垃圾MOTO Z2f则是不重置系统无信号流量，谁知道V30的电信可以电话短信，不可以流量！

    那么开始配置流量上网吧！

    工具
    链接:链接: https://pan.baidu.com/s/11crXTBU5w-SJrdj9gMCVDQ 密码: 6xvb

    1. 第一步：安装驱动，注意手机驱动只有LG Bridge自己更新的有用！！！！

      1. 安装驱动，建议装LG Bridge 链接：http://pan.baidu.com/s/1jHQryyq 或者直接LG官网下载LG Bridge

      2. 安装好QPST 链接：http://pan.baidu.com/s/1dEGsolb

    2. 第二步：开启Port check Test，勾选，设置为enable，我的US998直接手机拨号*#546368#*998# 就会弹出了

      点一下Port check test 让下面的disabled 变成 enabled.好了后退出就行

      提醒：但是不同的机型可能进入工程模式不一样，而且路径可能稍微有点不同。可以一个一个主目录找，直到找到Port check Test，建议不要乱动里面的数据。

方法1：输入3845#*手机型号#，比如说手机是G6美版H993，输入3845#*993#（xxx(一个个找)-Port check Test-enable）

方法2：输入*#546368#*手机型号#，以作者LG V30+ US998为例： 输入*#546368#*998#可以成功进入工程模式（SVC menu-Port check Test-enable）

方法3：网络选择:输入5457#*手机型号#，比如说手机是G6美版H993，输入5457#*993#

方法4：输入##7764726996（其他版本如US998，后面三位数字就变成998）按拨号键，输入000000，进入隐藏菜单；点击W2BI-TEST SETTING选择W2BI SETTING ENABLE；（据反馈不适合ls998和as998因为这机子初始spc不是000000，spc码破解

以上任何方法只要开启开启Port check Test，勾选，设置为enable。即可。

    3. 第二步：QPST设置

      1. 手机连接电脑，下拉屏幕usb模式选择文件传输
      2. 从开始菜单中，打开QPST configuration。
      3. 点开Ports标签，选择右下角的add new ports，然后去掉√，选择带有unknown那个端口，再点“ok”按钮
      4. 点Start Client菜单中的EFS Explorer选项。
      5. 什么都不需要改，直接点OK按钮。

        等待设备响应。（大约1分钟）
      6. 找到/policyman目录，ctrl+a 全选里面的文件，点击右键选择delete删除，然后选择需要复制进去的文件，联通和电信分别选择对应的参数拖到里面。
            找到policyman文件夹，先备份原carrier_policy.xml: 鼠标右键，copy data file to PC。把修改好的carrier_policy.xml拖进policyman文件夹替换原文件；

    4>关闭QPST，重启手机，去手机设置→网络→更多→移动网络→接入点名称，添加中国电信APN，名称和APN都设置为CTNET即可。
    APN设置：
    名称：CTLTE 4G
    接入点名称:ctnet
    端口：80
    用户名：vnet@mycdma.cn
    密码：vnet.mobi
    服务器地址：10.0.0.200
    MMSC :http://mmsc.vnet.mobi
    彩信代理：10.0.0.200
    端口：80
    MCC:460 MNC:11
    身份验证类型：pap或chap
    APN类型：default,mms,supl,hirpi
    APN协议：IPv4
    APN漫游协议：IPv4
    彩信网络设置 ：
    名称：MMSC
    Apn：ctwap
    MMSC：http://mmsc.vnet.mobi
    MMS proxy：10.0.0.200
    port：80
    Apn Type：mms
    ```
    7. 添加完毕，点phone选择reset phone按钮，手机会重启，然后关闭QPST。
    8. 重启手机后，全球模式或者lte/cdma，然后接入点设置成ctnet （上网设置 ：设置-更多-移动网络-接入点-新建 电信接入点：名称和apn都为ctnet，保存退出） 就应该可以上4G了。
    9. 教程完毕，谢谢！
