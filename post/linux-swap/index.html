    • 修改linux的swap缓存
    2020-05-14 / 1 min read

    添加新的 swap 分区，比如默认阿里云是没有swap分区的
    1.创建文件夹 mkdir /my_swap/
    2.设置交换分区大小(4GB) dd if=/dev/zero of=/my_swap/swap bs=1M count=4096
    3.创建交换分区 mkswap /my_swap/swap
    4.启动交换分区 swapon /my_swap/swap
    5.若要想使开机时自启用，则需修改文件/etc/fstab中的swap行 /my_swap/swap swap swap defaults 0 0
    6.查看创建好的分区 free -h
    参考链接：https://blog.csdn.net/qq_33666373/java/article/details/78625982

    设置缓存比例 sysctl vm.swappiness=60 设置比例为60%
    sysctl vm.swappiness=0 使用比例0%，全部使用内存
    sysctl vm.swappiness=100 使用比例100% 全部使用物理空间，不使用内存
    修改 /etc/sysctl.conf 永久保存
    sysctl -p 刷新配置
